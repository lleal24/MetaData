1. RELEASE NOTES

JxDocument is a cross-platform library for viewing PDF document in Java Swing applications.
JxDocument allows loading PDF documents, receiving information about pages, text on pages and
render each page into an image that can be displayed in Java Swing applications.

To see the functionality of JxDocument in action run the JxDocument Demo
application that is available in JxDocument distribution package.

JxDocument API JavaDoc documentation and samples that show how to use JxDocument
API in Java applications are available in the distribution package.

================================================================================
2. GETTING STARTED

The JxDocument package contains following:
 - JxDocument library (/lib/jxdocument.jar)
 - JxDocument public API JavaDocs (/doc/javadoc)
 - JxDocument samples (/samples)
 - JxDocument Demo application (/demo)
 - JxDocument agreement License (License agreement.txt)
 - This Readme file (Readme.txt)

Installing Library JAR Files
----------------------------
All the JAR files in the 'lib' directory should be in Java application class path.
